About EcoHarvest

How the AI works

A deep dive into the dataset, models, and technology powering EcoHarvest.

Process

From soil data to recommendation

1. Data Input

You provide 7 parameters: N, P, K (soil nutrients), temperature, humidity, pH and rainfall. These represent the key agronomic factors that determine crop suitability.

2. Feature Scaling

Inputs are standardised using StandardScaler — transforming values to a common scale so the model is not biased by units (e.g. kg/ha vs °C).

3. AI Prediction

Our champion Random Forest model (99.32% accuracy) classifies the input into one of 22 crop categories using an ensemble of decision trees.

4. Result Delivery

The predicted crop name is decoded from the label encoder and returned to you instantly with a contextual icon and confidence indicator.

Model Performance

3 models trained — 1 champion selected

We benchmarked Random Forest, XGBoost & SVM. The model with highest validation accuracy is automatically saved as the prediction engine.

Random Forest

An ensemble of 100 decision trees, each trained on a random bootstrap sample of the data. Trees vote on the final class — majority wins. This bagging approach dramatically reduces variance and overfitting compared to a single tree.

Dataset

Training data at a glance

2,200 Samples

100 samples per crop across 22 varieties, ensuring balanced class distribution.

7 Features

N, P, K, Temperature, Humidity, Soil pH, Rainfall — the standard agronomic parameter set.

22 Classes

Grains, legumes, fruits, fibres and beverages — a comprehensive crop taxonomy.

80/20 Split

80% training, 20% validation. Random state fixed for full reproducibility.
